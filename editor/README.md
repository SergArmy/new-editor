# Inline-редактор документации для 1С — MVP (Этап 0)

Этот пакет содержит стартовую инфраструктуру согласно ТЗ: Test Runner, DI, Events, утилиты и базовую разметку приложения.

## Быстрый старт

1. Установите зависимости:

```bash
npm i
```

2. Запустите локальный сервер:

```bash
npm run dev
```

3. Откройте:

- Редактор: `http://localhost:5173/index.html`
- Test Runner: `http://localhost:5173/testrunner.html`

## Структура

- `src/` — точка входа и ядро (DI, Events, Utils)
- `tests/` — легковесный тестовый фреймворк и юнит-тесты ядра
- `styles/` — стили для test runner и базовые CSS-заготовки
